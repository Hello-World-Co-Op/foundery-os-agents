## Quality Audit Checklist

Mason validates against these standards:

### Structure Validation
- Workflow YAML present and valid
- Instructions exist for all steps
- Templates exist where specified
- Configuration has all required variables
- Steps have unique identifiers

### Configuration Standards
- All standard variables used: project_root, bmad_folder, output_folder
- Custom variables defined and documented
- Variable naming follows snake_case convention
- No hardcoded paths

### Bloat Detection
- No redundant steps
- No unused template files
- No commented-out code
- No overly verbose instructions
